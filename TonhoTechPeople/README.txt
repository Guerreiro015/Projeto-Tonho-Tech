TONHO TECH People 0.6.2 - Refinamento do Painel de Administração

Arquivos alterados:
- src/pages/Admin.jsx
- src/styles.css

Como aplicar:
1. Copie a pasta src deste patch para a raiz do projeto TonhoTechPeople.
2. Confirme a substituição dos arquivos.
3. Execute npm run dev.
4. Teste o cadastro de um usuário RH/DP.

Melhorias:
- Cadastro novo oferece apenas RH/DP e Suporte.
- ADMIN só pode ser atribuído na edição.
- Regional aparece somente para Suporte.
- Modal profissional para edição.
- Exibição e cópia das credenciais após cadastro/reset.
- Botão Desativar/Ativar mais claro.
- Proteção contra desativar a própria conta.
