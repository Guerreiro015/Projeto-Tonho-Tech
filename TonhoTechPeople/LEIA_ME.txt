TONHO TECH People 0.7.3 — Login por sessão da janela

ALTERAÇÃO
- A sessão de autenticação passa a usar sessionStorage.
- Atualizar a página mantém o usuário conectado.
- Fechar a aba ou o navegador encerra a sessão local.
- Ao abrir o sistema novamente, o login será exigido.
- Uma sessão antiga salva em localStorage é removida automaticamente.

INSTALAÇÃO
1. Copie a pasta src deste patch.
2. Cole na raiz do projeto TonhoTechPeople.
3. Confirme a substituição do arquivo.
4. Execute npm run dev e faça o teste descrito pelo assistente.

Não há migration nem novo deploy de Edge Function nesta atualização.
