# Arquitetura

A versão 2.0 marca o início da transição para uma arquitetura modular:

- `core/`: inicialização, rotas e regras centrais;
- `services/`: integração com dados e serviços;
- `app/`: áreas funcionais do produto;
- `components/`: componentes reutilizáveis;
- `brand/`: identidade visual;
- `docs/`: documentação técnica e de produto.
