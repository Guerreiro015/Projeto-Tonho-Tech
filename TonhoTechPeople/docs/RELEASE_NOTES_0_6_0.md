# TONHO TECH People Web 0.6.0 — Security Foundation

- Login por nome de usuário e senha, sem lista de contas.
- Sessão real pelo Supabase Auth.
- Administração exclusiva para perfil ADMIN.
- Criação e alteração de contas por Edge Function protegida.
- Vínculo obrigatório do Suporte com uma regional.
- RLS no banco para restringir colaboradores por regional.
- Suporte visualiza apenas as próprias solicitações.
- Bloqueio/ativação e redefinição de senha pelo Administrador.
- Remoção do PIN como mecanismo de autenticação.
