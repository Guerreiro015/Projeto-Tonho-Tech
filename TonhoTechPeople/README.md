# TONHO TECH People Web 0.4.3

Versão com edição de credenciais de usuários.

## Destaques

- Login conectado ao Supabase.
- Usuários administrados no Supabase.
- Admin pode editar credenciais dos usuários:
  - Nome
  - Usuário/Login
  - PIN
  - Perfil
  - Regional do suporte
  - Status ativo/inativo

## Executar

```bash
npm install
npm run dev
```

Acesse o endereço exibido pelo Vite, normalmente:

```text
http://localhost:5173
```
